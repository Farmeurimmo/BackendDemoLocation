# Backend site Demo location

Technologies utilisées :
- Spring Boot
- Spring Data JPA
- Spring Security
- MySQL (with HikariCP)
- Lombok

## Développement

```bash
# remplacez les valeurs par les vôtres
export DB_URL="jdbc:mysql://localhost:3306/backenddemolocation"
export DB_USERNAME="root"
export DB_PASSWORD="123456789"

./gradlew bootRun
```
